## Overview
This project is an interactive Power BI dashboard built to analyze Spotify songs and artists data.  
The goal is to explore listening trends, artist dominance, song popularity, and release patterns over time using clean, intuitive visuals.

---

## Key Metrics
- **789** Distinct Songs  
- **342** Artists  
- **3.28 mins** Average Song Duration  
- **89.62** Average Popularity Score  

---

## Dashboard Features
- **Songs & Artists Panel**  
  Quick selection of songs with album artwork for better context.

- **Songs by Album Type**  
  Distribution across singles, albums, and compilations.

- **Explicit vs Non-Explicit Analysis**  
  Comparison of content types in the dataset.

- **Songs by Year**  
  Trend analysis for releases across different years.

- **Average Popularity by Month / Quarter**  
  Seasonal popularity trends with toggle functionality.

- **Distinct Songs by Month**  
  Release frequency trends across the year.

- **Top Artists & Most Popular Songs**  
  Ranked views based on song count and popularity.
